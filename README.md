## Desafio proposto:
## Criar uma aplicação que utilize a api https://pokeapi.co/ para:
Mostrar uma lista de pokémons com FlatList em React Native exibindo:
exibindo:
  - Nome
  - Imagem
  - Tipo principal
    
Funcionalidades extras implementadas:
  - Pesquisar Pokémons pelo nome.
  - Tratamento dos estados de carregamento e de erros.
